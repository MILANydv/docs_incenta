# Incenta Documentation

Welcome to the Incenta API documentation. Build referral programs, track conversions, and manage payouts with simple REST APIs and webhooks.

## Getting Started

Start by [creating an account](https://incenta.dev/signup) and grabbing your API key from the dashboard.

- [Quickstart](/docs/quickstart) – Make your first API call in under 5 minutes.
- [Authentication](/docs/auth) – API keys, OAuth 2.0, and security best practices.
- [API Reference](/docs/api-reference) – Full endpoint catalog with request/response schemas.
- [Webhooks](/docs/webhooks) – Real-time event notifications for conversions and payouts.
- [SDKs](/docs/sdks) – Client libraries for popular languages.
- [Examples](/docs/examples) – Common integration patterns and use cases.

## API Base URL

```
https://api.incenta.dev/v1
```

All requests must include your API key in the `Authorization` header:

```
Authorization: Bearer inc_live_xxxxx
```
