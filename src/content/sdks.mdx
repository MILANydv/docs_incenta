# SDKs

Incenta provides first-party SDKs for popular languages.

## Node.js / TypeScript

```bash
npm install incenta-api
```

```js
import { IncentaClient } from 'incenta-api'

const client = new IncentaClient({ apiKey: process.env.INCENTA_API_KEY })

// Create a referral link
const link = await client.links.create({
  campaignId: 'camp_abc123',
  destination: 'https://example.com',
})
```

## Python

```bash
pip install incenta
```

```python
from incenta import IncentaClient

client = IncentaClient(api_key="inc_live_xxxxx")

link = client.links.create(
    campaign_id="camp_abc123",
    destination="https://example.com"
)
```

## Go

```bash
go get github.com/incenta/incenta-go
```

```go
import "github.com/incenta/incenta-go"

client := incenta.NewClient("inc_live_xxxxx")

link, err := client.Links.Create(ctx, &incenta.CreateLinkRequest{
    CampaignID:  "camp_abc123",
    Destination: "https://example.com",
})
```
